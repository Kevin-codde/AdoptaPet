## 1. Descripción del proyecto
AdoptaPet es una aplicación móvil orientada a facilitar la publicación y consulta de mascotas disponibles para adopción. La plataforma permitirá a los usuarios registrar información básica de una mascota que necesita un nuevo hogar y publicarla dentro de la aplicación para que otros usuarios puedan visualizarla.  
La aplicación ofrecerá una interfaz sencilla que permita explorar las mascotas publicadas, consultar sus detalles y acceder a la información de contacto del responsable de la publicación para gestionar el proceso de adopción fuera de la plataforma.  
El sistema se enfocará en la simplicidad de uso y en la facilidad para publicar y visualizar mascotas disponibles.
