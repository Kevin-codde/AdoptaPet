## 1. Descripción del proyecto

AdoptaPet es una aplicación móvil desarrollada con el propósito de facilitar el proceso de publicación y consulta de mascotas que se encuentran disponibles para adopción. La plataforma está dirigida a personas que desean encontrar un hogar responsable para sus mascotas o para animales rescatados, así como a usuarios interesados en adoptar y brindarles una nueva oportunidad.

La aplicación permitirá a los usuarios registrarse e iniciar sesión de manera segura para acceder a las funcionalidades principales del sistema. Una vez autenticados, podrán publicar mascotas ingresando información detallada como el nombre, tipo de animal, edad aproximada, descripción, fotografía y datos de contacto del responsable. Esta información será almacenada en una base de datos y estará disponible para ser consultada por otros usuarios de la aplicación.

En la pantalla principal, los usuarios podrán explorar un listado de mascotas publicadas, visualizar fotografías y conocer información básica de cada una. Además, tendrán la posibilidad de acceder a una vista detallada donde se mostrará la descripción completa de la mascota, los datos del propietario o responsable y la información necesaria para establecer contacto y coordinar el proceso de adopción.

La aplicación también contará con un módulo de perfil de usuario, desde el cual cada persona podrá consultar sus datos personales, visualizar las mascotas que ha publicado y eliminar aquellas publicaciones que ya no estén disponibles. Esto permitirá mantener actualizada la información mostrada en la plataforma.

El sistema utilizará tecnologías modernas de desarrollo móvil, incluyendo Android Studio con Kotlin, arquitectura MVP (Model-View-Presenter), Firebase Authentication para la gestión de usuarios, Firebase Storage para el almacenamiento de imágenes y Firestore o Room para la persistencia de datos. Esta combinación de herramientas garantizará una solución robusta, escalable y de fácil mantenimiento.

AdoptaPet se enfocará en ofrecer una experiencia de usuario sencilla, intuitiva y eficiente, reduciendo las barreras tecnológicas para que cualquier persona pueda publicar una mascota o encontrar un animal para adoptar. De esta manera, la aplicación contribuirá a promover la adopción responsable y a disminuir la cantidad de mascotas en situación de abandono.
